# **App Name**: OfficeFlow

## Core Features:

- Invoice Management: Create, edit, view, delete, and download invoices in PDF format.
- Purchase Order Management: Manage purchase orders with similar functionality to invoices.
- Quotation Management: Generate and manage quotations, including expiry dates.
- User Authentication: Secure user login and logout functionality.
- Settings Management: Manage office details such as name, address, contact information, tax details, and bank details; enable uploading signature and company seal images.
- AI-Powered Content Extraction Tool: Leverage AI tool to extract details such as Total Price, Status, and dates from scanned invoice documents or images uploaded into the application.

## Style Guidelines:

- Primary color: Deep purple (#a855f7) for a professional and calming feel.
- Secondary color: Royal blue (#4169E1) for interactive elements.
- Background color: Very light grey (#F0F0F0), near white, to provide a clean and professional canvas.
- Gradient option: A subtle gradient from #a855f7 to #4169E1 could be used for headers or accents.
- Font pairing: 'Inter' (sans-serif) for body text, 'Space Grotesk' (sans-serif) for headlines. Note: currently only Google Fonts are supported.
- Use professional and clear icons from a consistent set (e.g., Material Design Icons) for navigation and actions.
- Maintain a clean and structured layout with consistent spacing and alignment.
- Use subtle animations and transitions for a smooth user experience when loading data or navigating between sections.
